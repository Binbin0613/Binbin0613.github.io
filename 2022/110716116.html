nvidia-smi

¶3、使用conda创建虚拟环境

1、打开Anaconda Prompt窗口，创建环境Pytorch

conda create -n pytorch python=3.6

2、激活环境

conda activate pytorch

3、查看环境当中具备哪些工具包：

pip list

4、查看pytorch是否可用

python
>>> import torch
>>> torch.cuda.is_available();
# 返回True表示本机GPU可被pytorch使用

5、删除虚拟环境

conda remove -n pytorch_env(虚拟环境名称)  --all

6、将pytorch安装到指定的目录下

查看anaconda文档
conda create help

安装虚拟环境到指定目录下
conda create --prefix=D:\Anaconda\setup\envs\BinPytorch python=3.8

查看当前的环境及地址
conda info --envs

出现以下环境：
labelImg                 C:\Users\Administrator\.conda\envs\labelImg
base                  *  D:\Anaconda\setup
pytorch                  D:\Anaconda\setup\envs\pytorch

7、创建好了虚拟环境之后，安装pytorch
pytorch安装路径
